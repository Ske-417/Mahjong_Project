# Mahjong_Project
麻雀を作りたいと思ったので。

## 麻雀ゲーム (Mahjong Game)

日本の麻雀（リーチ麻雀）のPython実装です。

### クイックスタート

```bash
# ゲームを起動
python mahjong_cli.py

# テストを実行
python test_mahjong.py
```

### 詳細な情報

詳しい使い方やドキュメントは [README_GAME.md](README_GAME.md) をご覧ください。

### 実装されている機能

- 麻雀牌の管理（萬子、筒子、索子、字牌）
- 配牌とツモ、打牌の基本フロー
- 和了判定（4面子1雀頭）
- 聴牌判定と待ち牌の表示
- 4人対戦のCLIインターフェース
