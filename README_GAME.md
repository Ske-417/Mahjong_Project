# 麻雀プロジェクト (Mahjong Project)

日本の麻雀（リーチ麻雀）のPython実装です。

## 概要

このプロジェクトは、日本の麻雀ゲームの基本的な実装を提供します。4人のプレイヤーでプレイでき、基本的な麻雀のルールを実装しています。

## 機能

- **牌の管理**: 萬子、筒子、索子、字牌の全136枚の牌
- **手牌管理**: プレイヤーの手牌の管理と操作
- **和了判定**: 基本的な和了形（4面子1雀頭）の判定
- **聴牌判定**: 聴牌状態の検出と待ち牌の表示
- **ゲーム進行**: 配牌、ツモ、打牌の基本的なゲームフロー
- **CLIインターフェース**: コマンドラインでプレイ可能

## ファイル構成

- `mahjong_tile.py` - 麻雀牌のクラス定義
- `mahjong_wall.py` - 牌山（ウォール）の管理
- `mahjong_hand.py` - 手牌の管理と和了判定
- `mahjong_game.py` - ゲームのメインロジック
- `mahjong_cli.py` - CLIインターフェース
- `test_mahjong.py` - ユニットテスト

## 使い方

### ゲームの起動

```bash
python mahjong_cli.py
```

4人のプレイヤー名を入力してゲームを開始します。

### ゲームの流れ

1. 各プレイヤーに13枚の牌が配られます
2. 順番にツモ（牌を引く）と打牌（牌を捨てる）を行います
3. 和了形（4面子1雀頭）が揃うと和了となります
4. 聴牌（あと1枚で和了）になると待ち牌が表示されます

### テストの実行

```bash
python test_mahjong.py
```

## 牌の表記

- **萬子** (まんず): 1萬 ～ 9萬
- **筒子** (ぴんず): 1筒 ～ 9筒
- **索子** (そうず): 1索 ～ 9索
- **字牌** (じはい): 東、南、西、北、白、發、中
